# Glyph - Admin Game Icons

Place transparent PNG icons in this folder.
They are served at: /game-icons/[filename]

## Currently used icons:

|------------------------|-----------------------|---------|

| valorant-iron.png      | Iron rank badge       | 48×48px |
| valorant-bronze.png    | Bronze rank badge     | 48×48px |
| valorant-silver.png    | Silver rank badge     | 48×48px |
| valorant-gold.png      | Gold rank badge       | 48×48px |
| valorant-platinum.png  | Platinum rank badge   | 48×48px |
| valorant-diamond.png   | Diamond rank badge    | 48×48px |
| valorant-ascendant.png | Ascendant rank badge  | 48×48px |
| valorant-immortal.png  | Immortal rank badge   | 48×48px |
| valorant-radiant.png   | Radiant rank badge    | 48×48px |
| valorant-unranked.png  | Unranked rank badge   | 48×48px |

## Where to get rank PNGs:
- Official Valorant media kit
- https://playvalorant.com/en-us/media/
- Community resources: https://github.com/nicholasgasior/valorant-ranks

## Recommended format:
- PNG with transparent background
- Minimum 96×96px for rank badges (displayed at 48px, 2x for retina)
- Use exact filenames above — the app references them by name

## Fallback:
If a file is missing, the app shows a colored rectangle with rank abbreviation.
No errors will occur — missing icons degrade gracefully.
